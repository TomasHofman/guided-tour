Deploy applications

Add deployments and assign them to server groups from Content Repository, where all deployment content is listed.

Assign deployments to server groups from Unassigned Content for those deployments that have not been assigned to any server groups.

Add and enable deployments from Server Groups for those deployments that have been assigned to one or more server groups.

Add a deployment to a server group by uploading a new deployment, choosing an existing deployment from the content repository, or creating an unmanaged deployment to be managed outside of WildFly.
